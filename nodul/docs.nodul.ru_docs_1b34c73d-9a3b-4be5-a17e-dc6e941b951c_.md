## Описание узла [​](https://docs.nodul.ru/docs/1b34c73d-9a3b-4be5-a17e-dc6e941b951c/\#%D0%BE%D0%BF%D0%B8%D1%81%D0%B0%D0%BD%D0%B8%D0%B5-%D1%83%D0%B7%D0%BB%D0%B0 "Прямая ссылка на Описание узла")

**SetVariables**\- узел типа действие, необходимый для ввода новой переменной в сценарий. Добавленную переменную в дальнейшем можно использовать в сценарии.

См. [Пример сценария с использованием SetVariables и оператора add](https://docs.nodul.ru/docs/1ebd5701-a882-4dea-b3e1-7ab1415e50d0)

подсказка

Добавленная переменная уникальна в рамках сценария и может быть изменена в ходе выполнения узлов сценария. Если два узла **SetVariables** расположены по порядку и оба определяют значение одной переменной, то итоговое значение для переменной будет устанавливать последний узел **SetVariables.**

## Настройка узла [​](https://docs.nodul.ru/docs/1b34c73d-9a3b-4be5-a17e-dc6e941b951c/\#%D0%BD%D0%B0%D1%81%D1%82%D1%80%D0%BE%D0%B9%D0%BA%D0%B0-%D1%83%D0%B7%D0%BB%D0%B0 "Прямая ссылка на Настройка узла")

Для настройки узла **SetVariables** необходимо заполнить пары полей Ключ-Значение.

- **(** **1** **)** **Ключ**\- поле для ввода наименования переменной;

- **(** **2** **) Значение**\- поле для ввода значения переменной.

![](https://docs.nodul.ru/img/notion/1087ea49-c9fe-4adb-8102-ba7bf3b96e89/Untitled.png)

Если узел **SetVariables** связан с узлом **Iterator** и выполняется последовательно несколько раз, то выходные данные узла отображаются с указанием **Итераций**. Каждой итерации соответствуют свои выходные данные.
